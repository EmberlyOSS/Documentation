## Unsure Which Plan You Need?

- **Just starting?** Try **Spark** (free)
- **Regular file sharing?** **Glow** is perfect
- **Large file uploads?** **Flare** has 2 GB files
- **Running a team?** **Blaze** has team features
- **Enterprise scale?** **Inferno** or **Ember**

See the [Plans Overview](/docs/getting-started/plans-overview) for more details about each tier.
