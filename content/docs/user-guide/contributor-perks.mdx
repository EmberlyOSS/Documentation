Thank you for contributing to Emberly's open source project! Contributor perks are **permanent and contribution-based** — the more lines of merged code you contribute, the higher your tier and benefits.

## How to Unlock

**1. Contribute to Emberly**

- Fork [github.com/EmberlyOSS/Website](https://github.com/EmberlyOSS/Website)
- Make your contributions and submit pull requests
- Only **merged** pull requests count toward your LOC total
- Lines _added_ count (not deleted or modified-unchanged)

**2. Link your GitHub account**

1. Go to **Dashboard → Profile → Linked Accounts**
2. Click **"Link GitHub Account"**
3. Authorize the OAuth connection

**3. Perks are automatically applied**

Contributions are verified automatically across all Emberly repositories. Check your current tier in **Dashboard → Profile → Perks**.

## FAQ

**Do deletions count against my total?**
No. Only lines added in merged pull requests count.

**Does documentation count?**
Yes — any merged contribution to an Emberly repository counts.

**Can I reach multiple tiers over time?**
Yes. Perks stack as you cross each milestone.

**What counts as an "Emberly repository"?**
Any repository under the [EmberlyOSS](https://github.com/EmberlyOSS) organization.
