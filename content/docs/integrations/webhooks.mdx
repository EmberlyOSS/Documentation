Webhooks allow your application to receive real-time notifications when events occur on Emberly.

---

## What are Webhooks?

Instead of polling the API repeatedly to check for changes, webhooks send HTTP POST requests to your server whenever something happens on Emberly.

**Example:** When a user uploads a file, Emberly sends an HTTP POST to your webhook URL with details about the upload.

---

## Getting Started

### 1. Create a Webhook Endpoint

Create an HTTP endpoint on your server that accepts POST requests:

```javascript
// Node.js Express example
app.post('/webhooks/emberly', (req, res) => {
  const event = req.body;
  
  console.log(`Received event: ${event.type}`);
  console.log(`Data:`, event.data);
  
  // Verify signature (important for security!)
  if (!verifyWebhookSignature(req)) {
    return res.status(401).send('Unauthorized');
  }
  
  // Handle the event
  switch (event.type) {
    case 'file.uploaded':
      handleFileUpload(event.data);
      break;
    case 'file.deleted':
      handleFileDeleted(event.data);
      break;
    // ... more event types
  }
  
  // Acknowledge receipt
  res.status(200).json({ ok: true });
});
```

### 2. Register Webhook in Dashboard

1. Go to **Settings** → **Webhooks**
2. Click **Add Webhook**
3. Enter your endpoint URL: `https://yourapp.com/webhooks/emberly`
4. Select which events to receive
5. Copy your **Signing Secret** (use for verification)

### 3. Verify Webhook Signature

Always verify that webhooks come from Emberly:

```javascript
const crypto = require('crypto');

function verifyWebhookSignature(req) {
  const signature = req.headers['x-emberly-signature'];
  const timestamp = req.headers['x-emberly-timestamp'];
  const secret = process.env.EMBERLY_WEBHOOK_SECRET;
  
  // Prevent replay attacks (timestamp shouldn't be too old)
  const requestTime = parseInt(timestamp);
  const now = Math.floor(Date.now() / 1000);
  if (Math.abs(now - requestTime) > 300) {
    return false; // Request older than 5 minutes
  }
  
  // Verify signature
  const body = req.rawBody; // Get raw body as string
  const message = `${timestamp}.${body}`;
  const expectedSignature = crypto
    .createHmac('sha256', secret)
    .update(message)
    .digest('hex');
  
  return signature === expectedSignature;
}
```

```json
  "timestamp": "2024-01-15T10:30:00Z",
```

## Webhook Delivery

### Retry Policy

If your endpoint returns a non-2xx status code, Emberly retries with exponential backoff:

| Attempt | Delay | Total Wait |
|---------|-------|-----------|
| 1st | Immediate | 0 seconds |
| 2nd | 5 seconds | 5 seconds |
| 3rd | 30 seconds | 35 seconds |
| 4th | 5 minutes | 5 min 35 sec |
| 5th | 30 minutes | 35 min 35 sec |
| 6th | 2 hours | 2 hrs 35 min |
| 7th | 5 hours | 7 hrs 35 min |
| 8th | 10 hours | 17 hrs 35 min |

After 8 failed attempts, the webhook is disabled. You can re-enable it in your settings.

### Response Requirements

**Always respond with:**
- HTTP status `200` - `299` (success)
- Within **30 seconds**
- Keep response body under 1 MB

**Examples of valid responses:**

```json
{ "ok": true }
```

```json
{ "received": true }
```

```
OK
```

Any response code outside 200-299 is treated as failure and triggers retries.

## Best Practices

### 1. Verify Signatures

Always verify webhook origin:

```python
def verify_signature(request, secret):
    signature = request.headers.get('X-Emberly-Signature')
    timestamp = request.headers.get('X-Emberly-Timestamp')
    
    if not signature or not timestamp:
        return False
    
    # Check timestamp freshness
    if abs(time.time() - int(timestamp)) > 300:
        return False
    
    # Verify signature
    message = f"{timestamp}.{request.data.decode()}"
    expected = hmac.new(
        secret.encode(),
        message.encode(),
        hashlib.sha256
    ).hexdigest()
    
    return hmac.compare_digest(signature, expected)
```

### 2. Idempotency

Events might be delivered multiple times. Handle duplicates gracefully:

```javascript
async function handleWebhook(event) {
  // Check if we already processed this event
  const existing = await db.events.findOne({ eventId: event.id });
  if (existing) {
    console.log(`Event ${event.id} already processed`);
    return { ok: true };
  }
  
  // Process the event
  await processEvent(event);
  
  // Mark as processed
  await db.events.insert({ eventId: event.id, processedAt: new Date() });
  
  return { ok: true };
}
```

### 3. Async Processing

Don't do heavy processing in the webhook handler. Queue the work:

```javascript
app.post('/webhooks/emberly', async (req, res) => {
  const event = req.body;
  
  if (!verifyWebhookSignature(req)) {
    return res.status(401).send('Unauthorized');
  }
  
  // Acknowledge immediately
  res.status(200).json({ ok: true });
  
  // Process async (don't await)
  queue.enqueue(() => processEvent(event));
});
```

### 4. Monitor Webhook Health

Track delivery status:

```javascript
// Log all webhook events
axios.interceptors.response.use(
  response => {
    logger.info('Webhook delivered', {
      eventId: response.data.id,
      status: 200
    });
  },
  error => {
    logger.error('Webhook failed', {
      eventId: error.response?.data?.id,
      status: error.response?.status
    });
  }
);
```

### 5. Webhook Signature Secret Management

Treat your signing secret like a password:

```javascript
// Good: Use environment variable
const SECRET = process.env.EMBERLY_WEBHOOK_SECRET;

// Bad: Hard-coded in code or version control
const SECRET = "abc123def456"; // Don't do this!
```

## Common Webhook Use Cases

### 1. Auto-delete Expired Files Locally

```javascript
app.post('/webhooks/emberly', async (req, res) => {
  if (req.body.type === 'file.deleted') {
    const { fileId } = req.body.data;
    
    // Clean up from your system
    await localDb.files.delete(fileId);
    console.log(`Deleted file ${fileId} from local cache`);
  }
  
  res.status(200).json({ ok: true });
});
```

### 2. Send Notification on Upload

```javascript
app.post('/webhooks/emberly', async (req, res) => {
  if (req.body.type === 'file.uploaded') {
    const { name, url } = req.body.data;
    
    // Notify user
    await slack.send({
      channel: '#uploads',
      text: `New file uploaded: ${name}`
    });
  }
  
  res.status(200).json({ ok: true });
});
```

### 3. Trigger Workflow on Plan Upgrade

```javascript
app.post('/webhooks/emberly', async (req, res) => {
  if (req.body.type === 'user.upgraded') {
    const { userId, newPlan } = req.body.data;
    
    // Trigger welcome email for new features
    await sendEmail(userId, {
      template: `welcome-${newPlan}`,
      subject: `Welcome to ${newPlan.toUpperCase()} plan!`
    });
  }
  
  res.status(200).json({ ok: true });
});
```

### 4. Sync User Data

```javascript
app.post('/webhooks/emberly', async (req, res) => {
  if (req.body.type === 'user.created') {
    const { userId, email } = req.body.data;
    
    // Sync to your database
    await myDb.users.insert({
      emberly_id: userId,
      email: email,
      synced_at: new Date()
    });
  }
  
  res.status(200).json({ ok: true });
});
```

## Troubleshooting

### Webhooks Not Arriving

1. **Check endpoint is publicly accessible**
   ```bash
   curl -X POST https://yourapp.com/webhooks/emberly -d '{"test": true}'
   ```

2. **Check firewall/CORS** — Emberly sends from IP ranges (get list from dashboard)

3. **Check response code** — Make sure you return 200-299

4. **Check logs** — Review webhook delivery history in your settings

### Wrong Signature

- Verify you're using the correct secret from dashboard
- Use `req.rawBody` not parsed body for signature verification
- Include timestamp in signature calculation

### Webhook Disabled

If Emberly disables your webhook after repeated failures:
1. Fix your endpoint
2. Go to Settings → Webhooks
3. Click the disabled webhook
4. Click "Re-enable"


```json

  {
```

### Create Webhook

```bash
curl -X POST -H "Authorization: Bearer API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://yourapp.com/webhooks/emberly",
    "events": ["file.uploaded", "file.deleted", "user.upgraded"]
  }' \
  https://api.emberly.ca/webhooks
```

### Delete Webhook

```bash
curl -X DELETE -H "Authorization: Bearer API_KEY" \
  https://api.emberly.ca/webhooks/wh_abc123
```

```bash
curl -X POST -H "Authorization: Bearer API_KEY" \
  https://api.emberly.ca/webhooks/wh_abc123/retry/evt_xyz789
```
